# Support

## WordPress.org support

Use the WordPress.org support forum for:

- installation help
- configuration questions
- release support for the public plugin

## GitHub Discussions

Use GitHub Discussions for:

- feature ideas
- implementation questions
- legal and compliance edge cases
- workflow feedback
- non-technical product feedback

## GitHub Issues

Use GitHub Issues only for reproducible bugs and concrete development tasks.

Please include:

- WordPress version
- WooCommerce version
- PHP version
- active theme and active plugins
- exact steps to reproduce
- expected result
- actual result

## Simple feedback

If you are not comfortable writing a technical issue, use the feedback form in the plugin sidebar inside WordPress admin.

It is meant for:

- general impressions
- feature requests
- usability problems
- support questions from non-technical users

The admin feedback form stores entries locally in WordPress. Do not include passwords, licence keys, or customer personal data in that form.

## Deactivate vs uninstall

- Deactivating the plugin keeps your settings and stored data.
- Uninstalling removes plugin files.
- Data is deleted on uninstall only when the remove-data setting is enabled in the plugin settings.
