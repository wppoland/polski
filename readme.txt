== Description ==

**Polski for WooCommerce** adapts your online shop to all Polish legal requirements while adding powerful merchandising and sales features - completely free. One plugin, 40+ features, zero paywalls.

Whether you need GDPR consent checkboxes, Omnibus Directive 30-day lowest price display, double opt-in verification, right of withdrawal forms, or ODR dispute resolution notices - Polski handles it out of the box. On top of legal compliance, you get a full suite of ecommerce tools: wishlists, product compare, gift cards, subscriptions, bundles, pre-orders, and much more.

== Installation ==

= Automatic Installation =

1. Go to **Plugins > Add New** in your WordPress admin
2. Search for "Polski for WooCommerce"
3. Click **Install Now** and then **Activate**
4. Navigate to **Polski** in the admin menu to configure your settings

= Manual Installation =

1. Download the plugin ZIP file
2. Go to **Plugins > Add New > Upload Plugin**
3. Upload the ZIP file and click **Install Now**
4. Activate the plugin and configure it under the **Polski** menu

= Configuration =

After activation, visit the Polski settings page to:

* Enable the legal compliance features you need
* Configure GDPR checkboxes for checkout and registration
* Set up delivery times and unit prices
* Activate merchandising modules (wishlist, compare, bundles, etc.)

== Frequently Asked Questions ==

= Is Polski for WooCommerce free? =

Yes, completely free and open source under GPLv2. Every feature is included - there are no paywalls, premium tiers, or artificial limitations.

= Does Polski support GDPR compliance for Polish shops? =

Yes. Polski provides configurable legal consent checkboxes with a full audit trail, double opt-in registration, and data privacy controls - all designed specifically for Polish GDPR requirements (RODO).

= Does it support the Omnibus Directive (EU Directive 2019/2161)? =

Yes. Polski automatically tracks and displays the lowest price from the last 30 days on sale products, as required by the Omnibus Directive implementation in Polish law.

= What makes Polski different from other compliance plugins? =

Polski is the only WooCommerce plugin built specifically for the Polish market that combines full legal compliance (GDPR, Omnibus, withdrawal forms) with 40+ merchandising features (wishlists, bundles, gift cards, subscriptions) - all included for free in a single extension.

= Does Polski work with WooCommerce Blocks checkout? =

Yes. Polski fully supports both the classic and block-based checkout and cart, including legal checkboxes and all shopmark displays.

= Does Polski work with HPOS (High-Performance Order Storage)? =

Yes. Polski declares full compatibility with WooCommerce HPOS (Custom Order Tables).

= Can I use Polski with Przelew tradycyjny (bank transfer)? =

Yes. Polski includes a built-in pay by invoice (Przelew tradycyjny) payment gateway for traditional Polish bank transfers.

= Does Polski support Polish food labeling requirements? =

Yes. Polski includes dedicated fields for food products: ingredients, nutrition facts, and allergen declarations, in compliance with Polish food information regulations.

= Can Polski replace multiple separate plugins? =

Yes. Polski includes wishlists, product compare, quick view, waitlists, gift cards, product bundles, pre-orders, subscriptions, product badges, catalog mode, and more - features that normally require installing many separate premium plugins. Polski bundles them all for free.

= Is Polski compatible with my theme? =

Polski works with any WooCommerce-compatible theme. It uses standard WooCommerce hooks and templates, so it integrates seamlessly with themes like Storefront, Astra, Flatsome, OceanWP, and others.

= Is Polski compatible with WPML/Polylang? =

Yes, Polski includes compatibility layers for both WPML and Polylang.

= Does Polski support Polish translations? =

Yes. Polski ships with complete Polish translations (.po/.mo files) from day one. It is fully translatable via the WordPress.org translation platform.

= Where can I report bugs or suggest features? =

Please use the [WordPress.org support forum](https://wordpress.org/support/plugin/woocommerce-polski/) or visit our website at [wppoland.com](https://wppoland.com).
