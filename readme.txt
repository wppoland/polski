WooCommerce tools for Polish storefront, checkout and product information requirements.

== Description ==

Polski helps WooCommerce stores handle Polish checkout, pricing, legal and product information requirements in one place.

The free version includes:

* Unit prices, VAT notices and Omnibus price history
* Legal checkout checkboxes with consent logging
* Withdrawal flow with customer form and email confirmation
* Legal page generation and compliant order button text
* Product information tools, delivery times and GPSR-related data
* B2B, merchandising and storefront modules
* WooCommerce Blocks and HPOS support

== Installation ==

1. Upload the `polski` folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress
3. Open `Polski > Kreator` to complete the initial setup
4. Review `Polski > Moduły` and enable the modules you need
5. Check `Polski > Pulpit` to confirm the setup

== Frequently Asked Questions ==

= Does this plugin work with WooCommerce Blocks checkout? =

Yes, Polski fully supports both classic and block-based checkout.

= Is the Omnibus directive support automatic? =

Yes. Polski handles price tracking and display natively.

= Can I customize the checkbox labels? =

Yes. Built-in checkbox labels, descriptions and validation messages are configurable from the admin panel.

= Does it support HPOS (High-Performance Order Storage)? =

Yes, Polski is fully compatible with WooCommerce HPOS (Custom Order Tables).

= What PHP version is required? =

PHP 8.1 or higher.

= Is this plugin compatible with WPML/Polylang? =

Yes, Polski includes compatibility layers for both WPML and Polylang.
