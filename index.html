Polski for WooCommerce

This project does not ship a standalone page here. Vite only bundles assets that WordPress loads (build/admin.js, frontend entries, etc.).

To work on the admin React UI: run npm run build, activate the plugin in a local WordPress site, and open WP Admin → Polski. Use npm run dev only if you wire HMR against that site; opening this URL alone will not mirror wp-admin.

Gutenberg blocks are built by the second step of npm run build (webpack) into build/blocks/.
